# Proyecto Final - Análisis de Datasets - CODERHOUSE

Este proyecto incluye tres datasets organizados en carpetas:
- **Dataset1:** Información financiera trimestral.
- **Dataset2:** Datos de rendimiento académico.
- **Dataset3:** Información de productividad laboral.

Cada carpeta contiene:
1. Archivo CSV con los datos originales.
2. Notebook Jupyter con el análisis correspondiente.
3. Archivo TXT con información adicional sobre el dataset.
